Aplikacija „AutoShop“ razvijena je u ASP.NET Core MVC tehnologiji sa cistom slojevitom strukturom. Omogucava pregled i kupovinu automobila, upravljanje korpicom, kreiranje narudzbina i administraciju korisnika prema ulogama (Administrator, Kupac).

Cilj rada bio je demonstracija primene arhitekture ASP.NET Core u vidu funkcionalne e-commerce platforme za prodaju automobila. Aplikacija je organizovana u tri sloja: prezentacioni (MVC), poslovni (BLL) i sloj podataka (DAL), sto omogucava lako odrzavanje i prosirenje sistema.
